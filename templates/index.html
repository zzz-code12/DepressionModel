About This Project

This is a Student Depression Prediction System built using Machine Learning.

The model was trained on a dataset of 27790 students using Logistic Regression with 83.56% accuracy.

How it works:

Fill in the prediction form with your information and the model will predict if you are showing signs of depression.

Disclaimer:

This is not a medical diagnosis. Please consult a professional if you are experiencing mental health issues.
